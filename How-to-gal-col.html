How to install Galactic Colonies

First, you need to go to this link and download as ZIP https://github.com/tolek-rudnicki/tgame

Then go to the location that you want the game to be in and unzip the package

Open your terminal (ctrl+alt+t) and navigate location with the command cd

Then you need to do sudo apt-get install npm

Check if you have NODE installed by doing node -v

Then npm i if you have if not go to Node instllation

Run the following command: npm i -g typescript ts-node

Then you are redy to do ts-node/(Your folder with the game)tgame/src/index.ts
